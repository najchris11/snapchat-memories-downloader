Drop the Linux x64 binaries here as zip files:

  exiftool.zip  — must contain the exiftool executable at the zip root
  ffmpeg.zip    — must contain the ffmpeg executable at the zip root

Sources:
  exiftool: https://exiftool.org/  (source or your distro package, then rezip the binary)
  ffmpeg:   https://ffmpeg.org/download.html  (Linux Static Builds)

The app extracts these to ~/.snapvault/bin/ on first run and sets +x automatically.
