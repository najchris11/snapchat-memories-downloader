Drop the Windows x64 binaries here as zip files:

  exiftool.zip  — must contain exiftool.exe at the zip root
  ffmpeg.zip    — must contain ffmpeg.exe at the zip root

Sources:
  exiftool: https://exiftool.org/  (Windows Executable)
  ffmpeg:   https://ffmpeg.org/download.html  (Windows Builds → essentials build)

The app extracts these to %USERPROFILE%\.snapvault\bin\ on first run.
